In this blog post, we'll be discussing implementing an assortment of geometric modelling topics. Topics include implementing de Casteljau's algorithm to create Bézier Curves/Surfaces, and also working with the half-edge data structure to construct triangle meshes.

# Bézier Curves with 1D de Casteljau's Subdivision.
To begin with, we will start out by implementing Bézier curves using de Casteljau's algorithm. To recap, de Casteljau's algorithm works by using linear interpolation between our $n$ control points to find a new set of control points, and it recursively does this until a single point is obtained: this point is our evaluated point on the Bézier curve.

For the algorithm, we are given $n$ initial control points and some parameter $t \in [0, 1]$. Then, the intermediate control points of the next subdivision is calculated as follows:
$$
\begin{align*}
    p'_i &= (1-t)p_i + tp_{i+1}
\end{align*}
$$
